Program BAZA DANYCH


Prosta uruchamiana w konsoli baza danych. Baza działająca w pamięci.


Technologia:
  język: RUBY (wersja ruby 3.0.0p0)
  gemy wykorzystane w testach: rspec


Uruchomienie programu:
  Program uruchamiamy poprzez plik main.rb Plik znajduje się w katalogu lib.
  Po przejściu do katalogu lib uruchamiamy program przy pomocy komendy:

  ruby main.rb

Aby zakończyć działanie programu wpisz komendę koniec. Aby uzyskać pomoc wpisz komendę pomoc.
